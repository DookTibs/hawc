    Create data pivot

    Two different types of data-pivots can be created. Please select the type you'd like to use:

    Option #1: All endpoints (Query-based)

    Create a data-pivot by selecting all data of a particular study-design in the database. This is a full data-download of all available data in HAWC for a particular assessment and study-design, but can be filtered to include a smaller subset of data. Your visualization settings will always be applied to the dataset; therefore the visualization may change if additional data is added, but will always include the latest data from the HAWC database.

    Select Option #1

    Option #2: Subset of endpoints (Query-based)

    Create a data-pivot by selecting specific endpoints to be included into a data pivot. This design should be used to select a subset of endpoints a priori. Your visualization settings will always be applied to the dataset; therefore the visualization may change if the selected-endpoints change, but will always include the latest data from the HAWC database.

    Select Option #2

    Option #3: Input file (Input-file)

    Create a data-pivot based on an uploaded file of data. Your visualization settings will always be applied to the dataset; therefore the visualization may change if additional data is added, but will always include the latest data from the HAWC database.

    Select Option #3 {% endblock %}
